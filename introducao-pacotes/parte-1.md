# Módulos vs Pacote

- **Módulo**:
  - Objeto que serve como unidade organizacional do código que é carregado pelo comando import.
  - OBS.: Qualquer arquivo .py é um modulo
  -  ***Vantagens da modularização***:
- **Pacote**: coleção de módulos com hierarquia.
